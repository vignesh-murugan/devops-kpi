Measuring DevOps

DevOps and its KPI

DevOps

Collabration between Development and Operations to achieve a common business goal.

Common Goals

  • Release on Demand
  • Fail fast
  • Less recovery time
  • Easy adoption of new technology

Common Barriers

These barriers are known things and can be easily overcome to adapt DevOps in an organization.

Velocity

Measure of the rate at which an organization can deliver software change

  • Frequency of Deployment

    • Number of deploys

  • Frequency of Build

    • Number of automated builds

  • Speed of Testing

    • Number of automated testing

Quality

  • Deploy success rate

    • Number of failures in deploy

  • Build success rate

    • Number of failures in build

  • Number of Defects
  • Performance

Productivity

  • Tool usage

    • Identify the tools usage and replace with better

  • MTTR

    • Mean Time To Repair

Security

  • Code Quality Check
  • DevSecOps
